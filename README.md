# Bodleian Library Management System

## 📌 Description
This project is a relational database system designed to manage a library environment.  
It supports core library operations such as managing books, members, loans, reservations, and fines.

The system was developed as part of coursework to demonstrate database design, normalization, and SQL implementation.

---

## ⚙️ Technologies Used
- MySQL
- phpMyAdmin
- SQL (DDL & DML)

---

## 🗄️ Database Structure

The database includes the following main tables:

- Authors
- Books
- BookCopies
- Members
- Loans
- Reservations
- Fines
- Genres
- Publishers

### Key Features:
- Track book availability and copies
- Manage member records
- Handle borrowing and returning of books
- Record fines for overdue items
- Maintain relationships using foreign keys
